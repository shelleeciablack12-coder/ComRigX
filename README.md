# ComRigX Chat App

A simple real-time chat app using React, Node.js, Express, and Socket.IO.

## Features
- Ask for username on entry
- Join a single global chat room
- Send and receive messages in real time
- Show username, message text, and timestamp

## Getting Started

### Server
1. Open a terminal in the `server` directory.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the backend server (default port: 3001).

### Client
1. Open a terminal in the `client` directory.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the React app (default port: 3000).

Open [http://localhost:3000](http://localhost:3000) in your browser to use the chat app.
